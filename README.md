# Customer Segmentation & Retention Analysis (SQL)

## Overview
Repository ini berisi kumpulan mini project SQL yang berfokus pada analisis perilaku pelanggan
menggunakan Online Retail Dataset.

Analisis difokuskan pada pola transaksi dan repeat purchase untuk membantu pengambilan
keputusan bisnis terkait operasional, stok, dan strategi retensi customer.

## Projects

### 01. Transaction Day Analysis
**Objective:**  
Mengidentifikasi hari dengan jumlah transaksi tertinggi untuk mendukung perencanaan stok
dan efisiensi operasional.

**Key Insights:**
- Hari Jumat memiliki jumlah transaksi tertinggi.
- Hari Selasa dan Kamis juga menunjukkan volume transaksi tinggi.
- Hari dengan transaksi terendah adalah hari Minggu.

### 02. Repeat Purchase & Customer Segmentation Analysis
**Objective:**  
Menganalisis perilaku repeat purchase customer berdasarkan jumlah transaksi.

**Key Insights:**
- One-time buyer menyumbang transaksi tertinggi (432 transaksi).
- Medium repeat buyer menghasilkan 295 transaksi.
- High repeat buyer hanya menyumbang 79 transaksi.
